# Australia & New Zealand Travel Guide for Chinese Travelers

A practical, casual travel guide for Chinese-speaking first-time visitors to Australia and New Zealand.

It covers:

- airport arrival and customs basics
- public transport and crossing rules
- safety notes for nature spots and city walks
- emergency contacts
- everyday shopping: supermarkets, pharmacies, malls, electronics stores, and household basics

Start here:

- [中文 README](./README.zh.md)
- [Full Chinese guide](./澳大利亚-新西兰旅行攻略.md)

Images are stored in [`assets`](./assets) and linked with relative paths so they render on GitHub.

This is a living personal guide. Always double-check official immigration, transport, and local safety information before you travel.

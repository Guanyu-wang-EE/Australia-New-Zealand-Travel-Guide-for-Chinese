# 澳大利亚-新西兰旅行攻略

写给第一次去澳洲/新西兰的中文旅行者。语气偏实用和随手查，适合出发前快速过一遍，也适合落地后临时翻。

内容包括：

- 入境、申报和机场流程
- 公共交通、刷卡和过马路规则
- 自然景区和城市自由活动安全提醒
- 紧急电话和翻译求助
- 日常采购：超市、药房、商场、百货、数码家电
